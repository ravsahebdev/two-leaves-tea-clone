# 🍃 Two Leaves Tea Clone

A fully responsive clone of the Two Leaves Tea website built using HTML, CSS, and JavaScript.

This project was created as part of my frontend development learning journey while practicing responsive web design, modern UI layouts, animations, navigation systems, and large-scale website structuring.

---

## 🚀 Features

- Fully Responsive Design
- Sticky Navigation Bar
- Mobile Hamburger Menu
- Interactive Hero Section
- Product Showcase Cards
- Horizontal Product Sliders
- Reviews Section
- Discover Section
- Journal/Blog Section
- CTA Sections
- Footer with Newsletter Form
- Smooth Hover Effects
- Modern UI Layout

---

- Font Awesome
